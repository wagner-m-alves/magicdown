# magicdown
Baixador de episódios de séries. Desenvolvido utilizando a linguagem Python e Selenium. Este programa entra no site cadastrado e verifica se há episódios novos das respectivas séries/temporadas cadastradas nos respectivos arquivos contidos no diretório DATA. 
Existem dois sites cadastrados ComandoTorrentsOrg e PandaFiles. O site ComandoTorrentsOrg saiu fora do ar, assim, não é possível baixar deste site, mas existe um outro, que pode funcionar (comandotorrent.net - NÃO TESTADO).
Para baixar do site PandaFiles é necessário colocar o link inicial no arquivo N_WaitingToDownload. Para utilizar basta rodar a classe Main.
